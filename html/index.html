HTML meaning HyperText Markup Language. It's a programming language whitch used to describe the structure of web pages.
HTML can creat: lists, images, videos, text, headings, tables.
It can be easily said that html is a "skeleton". witch is "decorated" by css and given functionality by JS.(Look img)

Why HTML:

  1. HTML is actually considered one of the easiest programming languages to learn.
  2. With HTML anyone can build own website
  3. HTML can help you to learn other coding languages easier
  4. HTML is most used in front-end
